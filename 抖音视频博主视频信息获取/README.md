# 抖音视频博主数据采集工具

## 项目简介

这是一个基于Python和Playwright的抖音视频博主数据采集工具，用于采集抖音博主主页下的视频数据。该工具采用面向对象的设计模式，具有良好的可维护性和扩展性。

## 功能特点

- **面向对象设计**：采用模块化、面向对象的编程方式，代码结构清晰，易于维护和扩展。
- **抖音视频采集**：支持采集抖音博主主页下的视频数据，包括视频标题、链接、封面、点赞数等信息。
- **无限滚动加载**：模拟鼠标滚动操作，自动加载更多视频内容。
- **强大的反爬措施**：模拟人工操作，包括随机延迟、鼠标移动、随机用户代理等，有效避免被平台风控。
- **数据导出功能**：支持将采集的数据保存为Excel文件，并可选择在Excel中嵌入视频封面图片。
- **授权系统**：内置设备授权系统，保障软件安全使用。
- **用户友好界面**：提供直观的图形界面，操作简单易用。

## 技术栈

- Python 3.8+
- Playwright（浏览器自动化）
- pandas（数据处理）
- openpyxl（Excel操作）
- tkinter（图形界面）
- requests（网络请求）

## 安装说明

1. **克隆项目**

```bash
git clone https://github.com/yourusername/douyin-spider.git
cd douyin-spider
```

2. **安装依赖**

```bash
pip install -r requirements.txt
```

3. **安装Playwright浏览器**

```bash
playwright install
```

## 使用方法

1. **启动浏览器**

   - 首先需要使用以下命令启动Chrome浏览器（带调试端口）：
   
   ```bash
   chrome.exe --remote-debugging-port=9222
   ```
   
   - 或者创建一个Chrome快捷方式，在目标字段中添加 `--remote-debugging-port=9222` 参数。

2. **运行工具**

   ```bash
   python douyin_spider.py
   ```

3. **登录抖音**

   - 在启动的浏览器中打开抖音网站（https://www.douyin.com）
   - 手动完成登录操作（首次使用时需要）

4. **采集数据**

   - 在工具界面中，点击"启动浏览器"按钮连接到已启动的Chrome
   - 导航到目标博主的主页
   - 设置滚动次数（每滚动一次会加载更多视频）
   - 点击"开始采集视频数据"按钮开始采集

5. **保存数据**

   - 采集完成后，点击"手动保存数据"按钮将数据保存为Excel文件
   - 可选择"导入表格插入图片"功能在Excel中嵌入视频封面

## 注意事项

- **合规使用**：请遵守抖音平台的用户协议和robots协议，不要进行高频、批量的恶意爬取。
- **登录状态**：首次使用时需要手动登录抖音，登录状态会自动保存。
- **反爬措施**：工具已内置反爬措施，但仍需合理控制采集频率，避免被平台风控。
- **授权系统**：软件需要授权才能使用全部功能，可根据界面提示获取授权码。
